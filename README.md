# 🃏 皮克松赌场小游戏合集

> 以赌神高进的跟班、验牌师 **皮克松** 为原型的网页小游戏集合。
> 江湖气十足，输赢看淡，不服就干！

## 🎮 游戏列表

| # | 游戏 | 状态 |
|---|------|------|
| 1 | 🎲 **掷骰子** — 比大小，压大压小 | ✅ |
| 2 | ♠️ **21点** — 经典 Blackjack | 🚧 |
| 3 | 🃏 **验牌** — 考验眼力，找出老千牌 | 🚧 |
| 4 | 🎰 **老虎机** — 三列转盘，凑对赢钱 | 🚧 |
| 5 | 🀄 **猜大小** — 三颗骰子，猜点数范围 | 🚧 |

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript
- 无框架依赖，打开即玩
- 响应式设计，手机/电脑都能玩

## 🚀 启动

直接用浏览器打开 `index.html` 即可，或者：

```bash
npx serve .
```
